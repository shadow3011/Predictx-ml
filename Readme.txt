# PredictX (ML Commodity Predictor)

## Backend Setup
cd backend
pip install -r requirements.txt
python app.py

## Frontend Setup
cd frontend
npm install
npm start

## Features
- ML prediction
- JWT authentication
- Graph visualization


## To manage DB 
-python backend/db_manager.py create-tables
-python backend/db_manager.py list-users
-python backend/db_manager.py delete-all-users
